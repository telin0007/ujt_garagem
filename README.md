# ujt_garagem
script de garagem vRpex, pode ser usado a vontade e tambem pode ser modficado

Comandos
/garagem abre o painel que mostar todos os carros que voce tem na garagem 9 apos setar o local das vagas aperte "enter" para salvar
/garagemsetup  usado para abrir o criado de garagens
/garagemapagar usado para apagar garagem
